### 新电脑 Ubuntu 上手三步

```bash
# 1. 把整个项目目录传过去（scp / git clone / 直接复制）
scp -r G:/AI/monica/monica-proxy-master user@server:/opt/

# 2. 一键安装
cd /opt/monica-proxy-master/deploy/ubuntu
chmod +x *.sh
bash install.sh

# 3. 填 Cookie 后启动
sudo nano /opt/monica-proxy/config.yaml
sudo systemctl start monica-proxy
```
